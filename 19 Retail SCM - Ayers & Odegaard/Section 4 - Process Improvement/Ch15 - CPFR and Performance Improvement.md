## West Marine Case Study — The Full Transformation

West Marine: US marine supplies retailer (~400 stores, $700M revenue)

### As-Is (Before Transformation)
- 6 separate buying teams; no coordination
- 75,000 SKUs; poor fill rates; excess inventory in some DCs, stockouts in others
- No supplier visibility; manual reordering
- Separate systems per region; no integrated demand plan

### Barriers to Success
- Culture of siloed buying (each buyer owned "their" supplier relationships)
- Incentives not aligned with SC outcomes (buyers measured on gross margin %, not inventory turns)
- No cross-functional SC process ownership

### Destination (To-Be)
- Single integrated demand plan
- Vendor-managed replenishment for top suppliers
- Synchronised replenishment cycle — all suppliers on the same weekly order rhythm
- 20–30% reduction in inventory; dramatically improved fill rates

### Pathway to Change (5 Steps)
1. **Leadership commitment** — CEO and CFO aligned on SC transformation
2. **Pilot** — start with top 10 suppliers; prove the model
3. **Process redesign** — define new replenishment process before changing systems
4. **Technology** — implement demand planning system after process is clear
5. **Rollout** — expand to all suppliers on the new model

## Continuous Improvement Cycles in Retail SC

### PDCA (Plan-Do-Check-Act)
The universal improvement cycle:
- **Plan:** Define the problem; set target; design solution
- **Do:** Implement on small scale / pilot
- **Check:** Measure results vs target
- **Act:** Standardise if successful; revise if not

### DMAIC in Retail SC Context
| Phase | Retail SC Application |
|---|---|
| **Define** | Define the SC problem: stockout %, excess inventory, high return rate |
| **Measure** | Baseline current state: fill rate, inventory days, perfect order % |
| **Analyse** | Root cause: demand forecast error? supplier OTD? DC capacity? |
| **Improve** | Design and pilot solution: CPFR, new ROP, supplier synchronisation |
| **Control** | Monitor KPIs; create standard operating procedure; sustain gains |

### CPFR (Collaborative Planning, Forecasting, and Replenishment)
The industry standard for retailer-supplier collaboration on demand planning:

| Step | Activity |
|---|---|
| **1. Collaborate on strategy** | Agree on business goals, category roles, promotional calendar |
| **2. Create joint forecast** | Retailer shares POS data + promo plan; supplier shares capacity + lead times |
| **3. Identify exceptions** | Where does forecast significantly deviate from baseline? |
| **4. Resolve exceptions** | Agree on revised forecast; supplier confirms capability |
| **5. Order generation** | System generates replenishment orders based on agreed forecast |
| **6. Monitor and feedback** | Track actuals vs plan; adjust in next cycle |

**CPFR participants:** Retailer buying team + Demand planner + Supplier sales team + Supplier supply planner

**Result at West Marine:** Supplier fill rates improved from ~70% → >90%; inventory down 25%

## S&OP in the Retail SC Context (Ch 15.3)

S&OP brings together:
- Merchandise/buying plan (demand side)
- DC/logistics capacity plan (supply side)
- Finance/budget plan
- Supplier capacity commitments

Monthly cycle: Review past performance → Update demand forecast → Identify gaps → Agree supply plan → Update financial forecast
