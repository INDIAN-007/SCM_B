## GENCO Case Study — Rise of the Return Loop

GENCO (now FedEx Supply Chain) built a business around retail returns processing:
- Retailers were losing 5–15% of revenue in unmanaged returns
- Returns were treated as a cost centre; often just disposed of
- GENCO created a systematic returns processing model that recovered value
- Key insight: **Returns are a profit opportunity, not just a cost.**

---

## Types of Returns in Retail

| Type | Description | Recovery Option |
|---|---|---|
| **Customer returns** | Customer brings product back to store; reason varies | Restock, refurbish, liquidate, dispose |
| **Unsold merchandise** | End-of-season stock not sold; sent back to DC/vendor | Markdown, off-price channel, vendor RTV |
| **Damaged goods** | Damaged in transit or store | Claim on carrier/supplier; recycle |
| **Defective products** | Manufacturing defect; recall | Return to OEM; destroy with credit |
| **Excess inventory** | Over-ordered; slow-moving | Liquidate, donate, return to vendor |
| **Trade-in / upgrade** | Customer trades old for new | Refurbish, resell, recycle |

---

## Nine Opportunities in Return Loops

| # | Opportunity | Description |
|---|---|---|
| 1 | **Reduced returns** | Fix root cause (sizing info, product description) → fewer returns in first place |
| 2 | **Improved customer service** | Fast, easy returns = higher customer satisfaction and repeat purchase |
| 3 | **Collaboration with partners** | Supplier takes back unsold stock; joint responsibility for return flows |
| 4 | **Customer feedback** | Returns data = product quality and fit feedback; feed into buying and design |
| 5 | **Material source** | Returned product = raw material for remanufacturing or refurbishment |
| 6 | **Environmental mitigation** | Divert returns from landfill; reduce carbon footprint |
| 7 | **Additional business** | Off-price resale channel; outlet stores; online liquidation |
| 8 | **Cash-to-Cash cycle reduction** | Faster returns processing → faster credit → lower working capital tied up |
| 9 | **Process standardisation** | Standard returns process across all channels → lower handling cost |

## Returns Processing Flow

```
Customer Return → Receiving at DC → Condition Assessment → Decision
                                                              ↓
                                    ┌─────────┬─────────────┬──────────────┐
                                  Restock  Refurbish  Liquidate/Sell  Dispose
                                    ↓         ↓              ↓              ↓
                               Back to   Outlet / B-grade  Off-price    Landfill /
                               primary   channel           marketplace   Recycle
                               inventory
```

## Returns KPIs

| KPI | Formula | Target |
|---|---|---|
| **Return Rate %** | Returns / Total sales × 100 | < 5% (FMCG), < 20% (fashion), < 10% (general retail) |
| **Returns Processing Time** | Time from return receipt to decision + action | < 3 days |
| **Recovery Rate %** | Value recovered / Value of returned goods × 100 | > 60–70% |
| **Returns-to-Restock %** | Items restocked / Total returns × 100 | Maximise (quality goods) |
| **Disposal Rate %** | Disposed / Total returns × 100 | Minimise |

## Reducing Returns at Source

| Return Cause | Fix |
|---|---|
| Wrong size (fashion/apparel) | Better size guides; fit technology; virtual try-on |
| Product not as described | Better product content (photos, specs, dimensions) |
| Defective product | Supplier quality improvement; incoming inspection |
| Change of mind | Non-refundable deposit; rental model; delay return window |
| Ordering too many (deliberate) | Flag and limit return behaviour; restocking fee |

## Returns in Phoon Huat / Hotel Context

| Context | Returns Type | Approach |
|---|---|---|
| **Phoon Huat** | Expired perishables, damaged goods, over-ordered seasonal | Supplier RTV clause; markdown before expiry; donate near-expiry |
| **Hotel** | Damaged linen, MRO defects, expired food | Supplier credit; insurance claims; food bank donation |
| **Asian Paints** | Unsold branch inventory, wrong colour tinted | RTV for untinted; markdown tinted; depot consolidation |
