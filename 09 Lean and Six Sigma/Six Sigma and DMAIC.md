## What Is Six Sigma?

"Six Sigma is a set of business tools, statistical theory, and quality control knowledge that helps improve your business procedures. It has the capability of improving performance while decreasing process discrepancy." — Six Sigma Council

**Six Sigma goal:** Reduce defects to 3.4 per million opportunities (DPMO).

```
Sigma Level → DPMO
1σ  → 690,000
2σ  → 308,537
3σ  → 66,807
4σ  → 6,210
5σ  → 233
6σ  → 3.4
```

## Lean Six Sigma

Combines:
- **Lean** — eliminate waste, improve flow
- **Six Sigma** — reduce variation and defects

Together they address both efficiency (lean) and quality (Six Sigma).

## DMAIC Methodology

The most important Six Sigma improvement framework:

### Define
- Identify the process/parameter to improve
- Understand impact on business / customers (Voice of Customer — VOC)
- Define the team and timelines
- Create a project charter to track progress

### Measure
- Identify tools and techniques to gather data
- Define input, process, and output parameters (IPO)
- Establish baseline performance metrics

### Analyse
- Create problem statement
- Root Cause Analysis (fishbone, 5-Why, Pareto)
- Implement process controls to stabilise
- Analyse deviation from standards
- Identify scope of improvement

### Improve
- Brainstorm solutions
- Create improvement roadmap with expected outcomes
- Communicate changes to process stakeholders
- Implement the solution

### Control
- Verify results against targets
- Adjust course if required
- Record learning and share across organisation
- Update SOPs and control charts

## RASCI Matrix (Kuldeepak)

Used in DMAIC projects to clarify roles:

| Letter | Role | Description |
|---|---|---|
| **R** | Responsible | Person who acts and performs the task |
| **A** | Accountable | Person liable for the outcome (one person only) |
| **S** | Supporting | Assists the team; not directly accountable for milestones |
| **C** | Consulted | Expert asked for advice; two-way communication |
| **I** | Informed | Updated on progress; one-way communication |
