## Quick Reference — All Lean Tools (Kuldeepak Ch.6)

| Tool | Purpose | Description |
|---|---|---|
| **DMAIC** | Improvement methodology | Define → Measure → Analyse → Improve → Control |
| **RASCI** | Responsibility matrix | Responsible, Accountable, Supporting, Consulted, Informed |
| **FMEA** | Risk proofing | Occurrence × Severity × Detection = RPN |
| **Poka-Yoke** | Error proofing | Shut-Out, Control, Attention strategies |
| **Kaizen** | Continuous improvement | Change for the better; daily incremental improvements |
| **5S / 6S** | Workplace organisation | Sort, Set in Order, Shine, Standardise, Sustain, Safety |
| **JIT** | Inventory reduction | Material available exactly when needed |
| **Kanban** | Visual pull scheduling | Card/signal system triggers replenishment |
| **Fishbone / Ishikawa** | Root cause analysis | 6Ms: Man, Machine, Material, Money, Method, Environment |
| **Pareto** | Prioritisation | 80% of effects from 20% of causes |
| **SMED** | Changeover reduction | Single Minute Exchange of Die — reduce setup to <10 minutes |
| **OTED** | Ultra-fast changeover | One Touch Exchange of Die — single-step changeover |
| **SHABD** | Logistics responsiveness | Single Hour Arrangement Based on Demand — Kuldeepak original |
| **Takt Time** | Pace matching | Available time / Customer demand — set production pace |
| **Andon** | Visual alarm | Lights/sounds signal quality deviation; production stops |
| **Jidoka** | Autonomation | Machine detects defect and stops automatically |
| **Heijunka** | Production levelling | Smooth production rate; reduce MURA |
| **Gemba Walk** | Shop floor observation | Systematic planned walk to find inefficiencies and good practices |
| **Gembitsu** | Focus on actual thing | Observe the actual product/machine, not a report |
| **Genjitsu** | Actual facts | Decisions based on ground reality, not perception |
| **5-Why Analysis** | Root cause drilling | Ask "Why?" iteratively until root cause found |
| **Control Charts** | Process monitoring | Statistical process control (SPC) — track variation over time |

## SMED vs OTED

| Method | Target | Description |
|---|---|---|
| **SMED** | <10 minutes | Single minute doesn't mean 1 minute — single digit of minutes |
| **OTED** | Single touch | One-step die exchange with clamps instead of screws |

**SMED process:** Shift Internal Setup (line down) → External Setup (line running) → Reduce Internal Setup time

---

## Takt Time Formula

```
Takt Time = Available Production Time / Customer Demand per Period

Example:
  Demand = 50 units/day
  Available time = 8 hours - 30 min breaks = 7.5 hours = 450 minutes
  Takt Time = 450 / 50 = 9 minutes per unit
```
