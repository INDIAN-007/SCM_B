## KPI Framework — Four Categories

### Quality Indicators (How well?)

| KPI | Formula | Description |
|---|---|---|
| **Putaway Accuracy %** | Correctly placed items / Total items × 100 | % items placed in correct bin or location |
| **Picking Accuracy %** | Correctly picked lines / Total lines × 100 | % orders picked correctly (right item, right quantity) |
| **Manhour Loss** | Incidents/accidents leading to lost productive hours | Safety indicator |
| **Inventory Accuracy %** | System stock vs Physical stock match | WMS/BIN Card vs physical count |

### Responsiveness Indicators (How fast?)

| KPI | Formula | Description |
|---|---|---|
| **Warehouse Order Processing Time** | Time from order receipt to dispatch | Full cycle time in warehouse |
| **Putaway Time** | Time from staging area to storage location per consignment | Productivity of receiving team |
| **Dock-to-Stock Time** | Time from vehicle arrival to item available in system | End-to-end receiving efficiency |
| **Order TAT (Turnaround Time)** | Time to pick, pack, load, and document an order | Internal fulfilment speed |

### Financial / Cost Indicators

| KPI | Formula | Description |
|---|---|---|
| **Total Warehousing Cost** | Sum of all warehouse expenses for period | Rent, labour, utilities, insurance, security, admin |
| **Product Damages In-House %** | Damaged stock value / Total stock value × 100 | Efficiency of storage and handling practices |
| **Order Fulfilment Cost** | Total cost / No. of orders fulfilled | Per-order cost including labour, equipment, consumables |

### Productivity Indicators

| KPI | Formula | Description |
|---|---|---|
| **Storage Space Utilisation %** | Volume occupied / Total storage volume × 100 | Cubic index (volume of item × units × stacking norms) |
| **Units Handled per Hour** | Total units / Manhours | Labour productivity for inward and outward |
| **Non-Storage Area %** | Non-storage space / Total space × 100 | Aisle, charging, packing, meeting room — should be minimised |
| **Inventory Turns** | COGS / Avg Inventory | How often inventory is sold and replaced |
| **Perpetual/Periodic Counting Compliance** | Counts done / Counts scheduled × 100 | Adherence to cycle count plan |

## Benchmark Targets (General Industry)

| KPI | Good | Excellent |
|---|---|---|
| Picking Accuracy | > 98% | > 99.5% |
| On-time Dispatch | > 95% | > 99% |
| Inventory Accuracy | > 97% | > 99.5% |
| Space Utilisation | 75–85% | 85–90% |
