# Types of Inventory

> **Source:** Kuldeepak Ch.4 · Chopra Ch.11–12
> **Tags:** #inventory #types #definitions

---

## Kuldeepak Classification

### Major Categories (by material status)
| Type | Description |
|---|---|
| **Raw Material (RM)** | Input materials not yet processed |
| **Work-in-Progress (WIP)** | Semi-finished goods in production |
| **Finished Goods (FG)** | Ready-to-sell products |

### Minor Categories (by utilisation)
| Type | Description |
|---|---|
| **Consumables** | Oil, paper, cleaning supplies — used in operations |
| **MRO (Maintenance, Repair & Overhaul)** | Spare parts and maintenance items |

### Purpose-Based Categories
| Type | Purpose |
|---|---|
| **Safety Stock** | Buffer against demand or supply uncertainty |
| **Seasonal Stock** | Built in anticipation of demand peaks |
| **Pipeline Stock** | In-transit inventory |
| **One-Shipment Stock** | All material for a project shipped together |

---

## Chopra Classification (Ch.11–12)

| Type | Definition | Driver |
|---|---|---|
| **Cycle Inventory** | Avg stock between replenishments (Q/2) | Order quantity (EOQ) |
| **Safety Inventory** | Extra stock against uncertainty | Demand/supply variability, service level |
| **Seasonal Inventory** | Build-up before predictable demand peaks | Demand seasonality |
| **Pipeline / In-Transit** | Goods moving between SC stages | Lead time × demand rate |

---

## Inventory Costs

| Cost | Description |
|---|---|
| **Holding / Carrying** | Interest on working capital + space + insurance + obsolescence |
| **Ordering** | Cost per purchase order — fixed, regardless of quantity |
| **Shortage / Stockout** | Lost sales, emergency procurement, customer dissatisfaction |
| **Obsolescence** | Expired, damaged, or unsaleable inventory |
