# Pharma & Healthcare — SCM Overview

> **Tags:** #industry #pharma #healthcare #coldchain #SCM

---

## Industry Profile

| Attribute | Characteristic |
|---|---|
| **Product Nature** | Life-critical; cannot stockout; strict expiry and batch control |
| **Regulation** | WHO-GMP, USFDA, CDSCO (India), HSA (Singapore), GDP (Good Distribution Practice) |
| **Cold Chain** | Vaccines (2–8°C), Biologics (<-20°C or <-80°C) |
| **Lead Times** | API sourcing: 60–90 days; Finished goods: 30–60 days |
| **Key Risk** | Temperature excursion, counterfeiting, batch recall |
| **Companies** | Sun Pharma, Dr. Reddy's, Cipla, Biocon, Pfizer, Abbott |

---

## Core Methods

| Method | Application |
|---|---|
| **FEFO strictly enforced** | Batch expiry tracking mandatory; earliest expiry dispatched first |
| **VED Analysis** | Vital = life-saving drugs; never stockout regardless of cost |
| **Safety Stock (high)** | Pharma carries 30–60 days SS for critical molecules |
| **Serialisation** | Every pack has unique serial number for track-and-trace |
| **Batch Recall readiness** | Within 24 hours, identify all locations a defective batch has reached |
| **Vendor qualification** | Suppliers must be GMP-certified; re-qualified annually |
| **GDP compliance** | Temperature-monitored transport at every leg of the chain |

---

## Pharma KPIs

| KPI | Target |
|---|---|
| Cold Chain Breach % | 0% (zero tolerance) |
| Batch Recall Response Time | < 24 hours to full traceability |
| Expiry Write-off % | < 0.5% of inventory value |
| OTIF (On-Time In-Full) | > 99% for hospital/critical customers |
| Counterfeit Detection | 100% serialisation coverage |

---

## Tips

1. **Never compromise on VED-V items** — life-critical drugs must have redundant stock and supplier
2. **Cold chain monitoring** — IoT temperature loggers at every handoff point; automatic alerts
3. **Parallel sourcing** — for critical APIs, always maintain at least 2 qualified suppliers
4. **Expiry-based ordering** — order quantities aligned to consumption before expiry; never over-stock short shelf-life biologics
5. **GDP training** — every person touching pharma inventory (warehouse, transport, pharmacy) must be GDP trained
