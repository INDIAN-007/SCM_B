## Role in Supply Chain

Transportation moves product between stages of the supply chain. It is **one of the biggest cost factors** and directly affects responsiveness.

---

## Key Tradeoffs

| Fast Mode (Air, Express) | Slow Mode (Ocean, Rail) |
|---|---|
| High cost | Low cost |
| Low inventory needed | High inventory needed (long lead times) |
| High responsiveness | Lower responsiveness |
| Suitable: High value, urgent, perishable | Suitable: Bulk, commodities, low urgency |

---

## Key Metrics

| Metric | Description |
|---|---|
| Average inbound / outbound transport cost | Cost per unit shipped |
| Average incoming / outgoing shipment size | Influences cost per unit |
| Average inbound / outbound transport cost per shipment | Total vs per-unit analysis |
| Fraction transported by mode | Mode mix composition |
| On-time delivery % | Service level measure |
| Vehicle utilisation | Actual vs rated capacity |

---

## Design Options for Transportation Networks

| Network Type | Description | Best For |
|---|---|---|
| Direct shipping | Factory → Customer | Large, frequent orders |
| Direct with milk runs | One truck visits multiple stops | Small, frequent deliveries |
| Centralised DC | Factory → DC → Customers | Moderate variety / volume |
| Cross-docking DC | No storage at DC | Fast-moving products |
| Tailored network | Mix based on customer / product segment | Complex demand patterns |

## Mumbai Dabbawalas — Best Practice (Chopra)

- Same-day delivery network in Mumbai, India
- 5,000 workers deliver 200,000 lunches daily
- Sigma level equivalent to Six Sigma
- Pure pull system — no inventory, just sorting and routing
